# 🌤️ Weather-Forecast v2.0

> **Project Status:** Completed ✅  
> **Original Launch:** June 2025  
> **Modernization & Refactoring:** March 2026

### 🚀 About the Project
A modern, secure, and high-performance weather forecasting application. The core focus of this 2.0 update was a complete infrastructure refactoring to align with industry standards, ensuring data security and optimized performance.

### 🛠️ What’s new in v2.0?
* **Vite.js Modernization:** Migrated from a static environment to a fast, efficient build ecosystem.
* **Data Security:** Implemented secure API key management. No more exposed keys; everything is now handled via **Vercel** environment variables.
* **Clean Code:** Fully refactored `script.js` to enhance readability, removing technical debt and optimizing API consumption.
* **Dynamic UI:** Improved data rendering logic and state animations for a smoother user experience.

### ⚙️ Tech Stack
* **Vite** (Build Tool)
* **JavaScript (ES6+)**
* **OpenWeatherMap API**
* **HTML5 / CSS3** (Flexbox & Grid)
* **Vercel** (Cloud Deployment)
